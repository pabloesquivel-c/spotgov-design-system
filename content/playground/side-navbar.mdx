This is a ground-up side navigation, separate from the existing sidebar. It composes shared `components/ui/*` primitives including Avatar, Badge, CompactButton, and Dropdown.

## Three regions

The panel is a flat, full-height column with a right border only. **Header** is a clickable organization card that opens the workspace switcher. **Content** holds a Tools divider and six capability rows, with Search Agent selected by default; Notifications is always last. **Footer** is a clickable account card that opens settings, support, documentation, sign-out, version, and terms links.

## Wired for later phases

The consuming app owns navigation. `onOrganizationChange` receives a workspace ID. Account links can be passed via `accountMenuLinks`; actions use `onAccountMenuAction`, and sign-out uses `onSignOut`. `organizationPlan` provides the plan label shown in the header.
